<img width="1184" alt="스크린샷 2024-08-31 오후 10 02 05" src="https://github.com/user-attachments/assets/4cf45463-ba88-4285-963d-6eb11ae2f92d">

## [프로젝트 설명]

- ''RiseMate'는 개발자들을 위해 이력서 관리 및 공유 기능을 지원하는 플랫폼으로, AI 코칭이나 전문가의 도움을 통해 이력서를 개선할 수 있는 서비스를 제공합니다.
- 'RiseMate'라는 이름은 성장의 의미를 담은 'Rise'와, 성장을 돕는 동반자라는 의미의 'Mate'를 결합하여 탄생했습니다.
